DDoS Detector

In this small project to identify victims of DDoS attacks, we observe traffic received to a dark net over the course of ~7 hours from victims all over the world. The inferrance of the DDoS attacks was done using an algorithm which tracks packets received over a set time period and categorizes the types received and how many of each packet type for each victim. To read more about how this was achieve check out the FAQ.
